# smh-connect 🔗

**smh-connect**, sosyal medya platformlarındaki (Instagram, TikTok, YouTube) dinamik içerikleri API üzerinden çekerek web arayüzlerine entegre eden hafif bir veri köprüsüdür.

## 🚀 Özellikler
- **Dinamik Veri Akışı:** Manuel güncelleme gerektirmeden en son paylaşımları çeker.
- **Asenkron Mimari:** `async/await` yapısı sayesinde site performansını etkilemez.
- **JSON Tabanlı:** Verileri standart JSON formatında işleyerek her türlü UI ile uyumlu çalışır.

## 🛠️ Teknik Stack
- **Dil:** JavaScript (ES6+)
- **Yöntem:** Fetch API & REST Entegrasyonu
- **Mimari:** Modüler JavaScript Sınıfları (OOP)
